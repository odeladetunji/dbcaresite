# CareTeam-D Clean React Project

A Vite + React + TypeScript project for the cleaned CareTeam-D landing page.

## Requirements

- Node.js 18 or newer
- npm

## Start locally

```bash
npm install
npm run dev
```

Then open the local URL shown in the terminal.

## Build for production

```bash
npm run build
npm run preview
```
